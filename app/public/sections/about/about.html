Sobre o Projeto

O OpenPBL foi desenvolvido como trabalho de conlusão de curso para obtenção do grau de bacharel em Sistemas de Informação na Pontifícia Universidade Católica do Rio Grande do Sul - PUCRS, em 2015.

O objetivo do sistema é servir de ferramenta de apoio a professores que desejam fazer uso do Problem-based Learning (PBL) em atividades escolares, com o auxílio de ferramentas computacionais. Com o propósito de desenvolver uma ferramenta atrativa, leve e intuitiva, foram utilizadas tecnologias modernas, que proporcianam uma experiência de usuário de alta qualidade. Além disto, a arquitetura do OpenPBL foi estruturada de forma altamente escalável, para que a continuidade do projeto e futuros estudos com base na ferramenta sejam possíveis.

OpenPBL é um projeto livre, acesse o código fonte .

Tecnologias Empregadas

O OpenPBL foi construído em Node.js®, um ambiente multi-plataforma livre para a construção de aplicações web. Aplicações em Node.js são escritas em JavaScript e orientadas a eventos, e oferecem condições ideais para a construção de sistemas leves, eficientes e altamente escaláveis.

O front-end foi construído utilizando o framework Angular.js, atualmente mantido pela Google. Angular.js adapta e extende o HTML tradicional para entregar conteúdo dinâmico através de data-binding de duplo sentido, o que permite a sincronização automática de models e views. Como resultado, o framework reduz a necessidade de manipulação da DOM com o objetivo de melhorar a testabilidade e performance, além de facilitar o desacoplamento do cliente e servidor. Isto facilita o desenvolvimento paralelo, método fortemente utilizado durante a concepção do OpenPBL, através de uma API construída no Apiary.
