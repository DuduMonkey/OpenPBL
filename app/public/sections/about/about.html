Sobre o Projeto

O OpenPBL foi desenvolvido como trabalho de conlusão de curso para obtenção do grau de bacharel em Sistemas de Informação na Pontifícia Universidade Católica do Rio Grande do Sul - PUCRS, em 2015.

O objetivo do sistema é servir de ferramenta de apoio a professores que desejam fazer uso do Problem-based Learning (PBL) em atividades escolares, com o auxílio de ferramentas computacionais. Com o propósito de desenvolver uma ferramenta atrativa, leve e intuitiva, foram utilizadas tecnologias modernas, que proporcianam uma experiência de usuário de alta qualidade. Além disto, a arquitetura do OpenPBL foi estruturada de forma altamente escalável, para que a continuidade do projeto e futuros estudos com base na ferramenta sejam possíveis.

OpenPBL é um projeto livre, acesse o código fonte .

Tecnologias Empregadas

O OpenPBL foi construído em Node.js®, um ambiente multi-plataforma livre para a construção de aplicações web. Aplicações em Node.js são escritas em JavaScript e orientadas a eventos, e oferecem condições ideais para a construção de sistemas leves, eficientes e altamente escaláveis.

O front-end foi construído utilizando o framework Angular.js, atualmente mantido pela Google. Angular.js adapta e extende o HTML tradicional para entregar conteúdo dinâmico através de data-binding de duplo sentido, o que permite a sincronização automática de models e views. Como resultado, o framework reduz a necessidade de manipulação da DOM com o objetivo de melhorar a testabilidade e performance, além de facilitar o desacoplamento do cliente e servidor. Isto facilita o desenvolvimento paralelo, método fortemente utilizado durante a concepção do OpenPBL, através de uma API construída no Apiary.

A API construída para o desenvolvimento do OpenPBL pode ser acessada aqui.

A persistência dos dados do OpenPBL é realizada através do banco de dados NoSQL MongoDB. A utilização de um banco de dados NoSQL possibilita o rápido desenvolvimento da base de dados por não necessitar de um schema pré definido, e aumenta de forma significante a escalabilidade e performance do sistema. Atualmente, o banco de dados é provisionado através do Mongolab.

Outras tecnologias utilizadas pelo sistema são destacadas abaixo:

  • SVG - Scalable Vector Graphics (SVG) é um formato de imagem vetorizada baseada em XML, para a criação de gráficos bi-dimensionais com suporte à interatividade e animação.

  • SMIL - Synchronized Multimedia Integration Language (SMIL) é uma linguagem que permite a fácil sincronização de elementos multimídia para apresentação web e interação. É utilizada no OpenPBL para a animação de elementos SVG.

  • Grunt - Gerenciador de tarefas JavaScript

  • Bower - Sistema de gerenciamento de pacotes para programação de aplicações cliente.

Inspiração

A ideia para a construção do OpenPBL surgiu com base em pesquisa sobre a utilização de ferramentas computacionais na área de educação. O trabalho e ideologia de alguns autores e institutos serviram de inspiração, e destacam-se os listados abaixo:

  • John Dewey - Filósofo e psicólogo Americano, suas ideias influenciaram reformas educacionais e sociais.

  • Richard P. Feynman - Cientista e físico teórico Americano, visitou o Brasil na década de 50 para lecionar.

  • Edutopia - Portal educacional e comunidade online que tem o propósito de fomentar a utilização de metodologias educacionais que estimulam e inspirem os estudantes, uma iniciativa da George Lucas Educational Foundation.

  • Problem-Based Learning: What and How Do Students Learn? - Artigo publicado pela professora Cindy E. Hmelo-Silver.
